Algumas Tecnologias Usadas

Laravel

O Laravel é um framework de web PHP livre e de código aberto, criado por Taylor Otwell e destinado ao desenvolvimento de aplicativos da web seguindo o padrão arquitetônico do modelo-visão-controlador (MVC).

Bitcoin

O Bitcoin é uma rede mundial de criptografia e sistema de pagamento digital denominado a primeira moeda digital descentralizada, já que o sistema funciona sem um repositório central ou um administrador único.

Bootstrap

O Bootstrap é um conjunto de ferramentas de código aberto para desenvolvimento com HTML, CSS e JS.

Open Graph protocol

O protocolo Open Graph permite que qualquer página da Web se torne um objeto rico em um gráfico social. Por exemplo, isso é usado no Facebook para permitir que qualquer página da Web tenha a mesma funcionalidade que qualquer outro objeto no Facebook.
